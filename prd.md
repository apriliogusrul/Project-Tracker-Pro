Project Tracker Pro

1. Overview

A lightweight, dynamic project management tool designed specifically for student or developer. The objective is to track personal development projects ( App and Website ) through their specific, distinct lifecycles to increase productivity and ensure projects actually get finished.

2. Target Audience

Primary User: Student/Developer
Needs: Clear visual tracking of tasks, dynamic workflows based on project type, and low friction for adding new tasks.

3. Project Types & Dynamic Workflows

The core differentiator of this app is that the Kanban board columns changed based on the project type.

4. MVP Core Features 

Project Creation: Ability to create a new project, assign a title, description, and select the type ( App vs Web )
Dynamic Kanban Board: A board that automatically generates the correct workflow columns based on the project type selected.
Task Management: Ability to create tasks within a project, assign them to a specific workflow stage, edit them, and delete them ( CRUD Operation ).
Drag-and-Drop: Ability to easily move tasks between stages as work progeresses.
Dashboard Summary: A simple home screen listing all active projects and their completion status.

5. Future Enhancements 

Time Tracking per Tasks ( Pomodoro Integration )
Markdown Support in Task Description for Code Snippets.
